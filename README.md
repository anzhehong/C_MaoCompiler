## 咋运行
```
# cd 到某路径
$ git clone https://github.com/anzhehong/C_MaoCompiler
$ cd C_MaoCompiler
$ gcc -c AZHMaoCompiler.c
$ gcc -o demo AZHMaoCompiler.o
# 执行可执行文件运行.mao文件中的代码
$ ./demo demo.mao
$ ./demo demo2.mao
```
